Privacy Policy

三国汉字战

This Privacy Policy explains how 三国汉字战 collects, uses, stores, and shares information. This app is an educational game intended for general audiences, including children under 13. It is designed to minimize data collection and to let families use the app without creating an account.

Effective date: April 26, 2026

No Ads This app does not serve third-party advertising.
No Account Required Users can play without signing up or logging in.
Local First Progress is stored on the device unless optional cloud sync is used.
Child-Focused The app is designed to avoid unnecessary personal data collection.

1. Information We Collect

The app collects only the information needed to operate its educational and save features.

  • Player names entered by the user or a parent/guardian.
  • Gameplay progress and save data, including study progress, unlocked content, roster choices, difficulty settings, and campaign progress.
  • An optional Family Key chosen by the user for cloud save and restore.
  • Technical requests needed to load app files, fonts, and the optional cloud save service.

The app does not intentionally collect a child’s real name, email address, phone number, precise location, contacts, photos, microphone input, or payment information.

2. Information Stored on the Device

By default, the app stores data locally on the device using browser/app storage so progress can be saved between sessions. This may include:

  • Player profiles and chosen display names.
  • Character mastery and learning progress.
  • Hero and campaign progress.
  • Unlocked stories and settings.
  • The optional Family Key used for cloud sync.

3. Optional Cloud Sync

The app offers optional cloud save and restore using Firebase Realtime Database. Cloud sync is only used if a user or parent/guardian enters a Family Key and chooses to save or restore data.

  • The Family Key is a custom word or phrase chosen by the user.
  • Player names and save data may be uploaded to the cloud save service so progress can be restored across devices.
  • Cloud sync is optional. The app can be used without it.
Families who do not want any data sent to the cloud should leave the Family Key blank and avoid using the cloud save or restore buttons.

4. How We Use Information

Information is used only to operate and improve the app experience, including to:

  • Save and restore progress.
  • Support multiple player profiles on one device.
  • Enable optional cloud backup and restore.
  • Provide educational features such as speech playback and progress tracking.
  • Load app assets reliably and support offline caching.

5. What the App Does Not Do

  • It does not show interest-based or third-party ads.
  • It does not sell personal information.
  • It does not require account registration.
  • It does not request camera, microphone, contact list, SMS, call log, or location permissions.
  • It does not use analytics or profiling SDKs to track users across apps or websites.

6. Third-Party Services and Network Requests

The app may communicate with the following services in order to function:

  • Firebase Realtime Database for optional cloud save and restore.
  • Content delivery services used to load certain libraries and font resources.
  • Platform speech synthesis services provided by the operating system or browser for text-to-speech playback.

These providers may process technical data such as IP address, browser/app request headers, and connection metadata as part of normal network operations. Their handling of data is governed by their own policies.

7. Children’s Privacy

Because this app may be used by children under 13, we aim to limit data collection to what is necessary for gameplay and optional save features. We do not knowingly collect more personal information from children than is reasonably necessary to operate the app.

Parents and guardians can help protect children’s privacy by using a nickname instead of a real name and by avoiding optional cloud sync if they do not want save data stored remotely.

8. Data Sharing

We do not sell or rent user data. Data is shared only in the following limited situations:

  • With Firebase, if the optional cloud save feature is used.
  • With technical service providers that deliver app resources or speech functionality.
  • If required by law or valid legal process.

9. Data Retention and Deletion

Local data remains on the device until it is deleted by the user, the app’s reset functions, or device/app removal.

Users or parents/guardians can manage data by:

  • Using the in-app reset and save management controls.
  • Clearing the app’s local storage or app data from the device.
  • Removing the app from the device.
  • Avoiding or discontinuing use of optional cloud sync.

10. Security

We take reasonable steps to reduce unnecessary data collection and to keep stored information limited to what the app needs. However, no storage system or internet transmission can be guaranteed to be 100% secure.

11. International Processing

If optional cloud sync or third-party network services are used, information may be processed on servers outside the user’s country, depending on the infrastructure used by those providers.

12. Changes to This Policy

This Privacy Policy may be updated from time to time to reflect changes to the app or legal requirements. The updated version will be posted in this file with a revised effective date.

13. Contact

If you have questions about this Privacy Policy or want help with data deletion requests, please contact the developer using the contact details provided on the Google Play listing for this app.

This policy is intended to describe the current behavior of the app as implemented in this project.
